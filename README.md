# ClawdEngineer Project

This project is an open-source collection of tools and libraries crafted by Cleng.

## Setup

To run this project locally:

1.  Clone the repository
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
